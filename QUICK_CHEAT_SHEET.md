## 🚨 当出现问题时...

### "我不知道怎么做"
```
→ 第1步: 打开 IMPLEMENTATION_GUIDE.md
→ 第2步: 找到你的 Task 编号
→ 第3步: 按照"实施步骤"来
→ 第4步: 还是不行? 问技术主管
```

### "任务花费的时间超过了估计"
```
→ 第1步: 立即通知 PM/TL
→ 第2步: 说出具体障碍
→ 第3步: 寻求技术支持
→ 第4步: 调整计划
```

### "我发现了一个安全漏洞"
```
→ 第1步: 立即停止当前工作
→ 第2步: 通知技术主管
→ 第3步: 记录详细信息
→ 第4步: 优先处理 (P0)
```

### "性能没有改善反而更差"
```
→ 第1步: 使用 Chrome DevTools 分析
→ 第2步: 记录具体指标和对比
→ 第3步: 通知技术主管
→ 第4步: 回到之前的方案，找原因
```
