# 小伍同学的个人博客

## 项目概述

小伍同学的个人博客是一个基于WordPress开源框架构建的现代化博客系统，集成大模型AI能力，提供智能化的内容创作、管理和阅读体验。该博客支持多用户管理、文章发布与评论互动、3D图库展示、智能搜索、AI辅助写作等功能，致力于打造一个集内容创作、知识分享、AI赋能于一体的个人知识平台。

**目标用户**: 技术爱好者、学习开发者、个人知识管理者

**核心价值**:
- AI辅助创作：通过大模型提供智能排版、内容优化、代码生成等能力
- 多维内容展示：支持传统文章、3D图库等多媒体形式
- 智能体验：提供智能搜索、个性化推荐等AI增强功能
- 优美的界面：现代化UI设计，支持AI生成图片素材
- 多用户协作：支持管理员、编辑、访客等不同角色权限管理

## 技术选型

| 类别 | 选择 | 理由 |
|------|------|------|
| 内容管理系统 | WordPress 6.0+ | 成熟稳定、插件生态丰富、易于扩展 |
| 编程语言 | PHP 8.1+ | WordPress官方支持版本，性能优化 |
| 数据库 | MySQL 8.0+ | WordPress原生支持，成熟可靠 |
| 缓存层 | Redis | 提升访问速度，支持会话管理 |
| CDN存储 | 腾讯云COS/阿里云OSS | 静态资源加速，支持大文件存储 |
| Web服务器 | Nginx | 高性能反向代理，支持静态文件服务 |
| PHP处理 | PHP-FPM | PHP官方推荐的处理方式 |
| AI服务 | 大模型API（可配置） | 支持OpenAI、通义千问等多种模型 |
| 前端框架 | Vue.js 3 + Element Plus | 优美的后台管理界面 |
| 3D展示 | Three.js | 浏览器端3D图库渲染 |
| 图像生成 | AI图像生成API | 自动生成UI图标和素材 |

## 核心功能

### 文章发布
- **目的**: 支持作者创建、编辑、发布博客文章
- **描述**: 提供富文本编辑器，支持Markdown，集成大模型AI辅助排版、内容优化、代码高亮等功能

### 评论互动
- **目的**: 实现访客与作者的互动交流
- **描述**: 支持游客评论、用户登录评论，包含评论审核、回复、点赞等功能

### 3D图库
- **目的**: 展示3D模型和交互式图像
- **描述**: 基于Three.js的3D模型展示功能，支持模型上传、预览、交互操作

### 智能搜索
- **目的**: 提供快速、准确的内容检索
- **描述**: 集成大模型增强的全文搜索，支持语义搜索、联想推荐

### 订阅推送
- **目的**: 实现文章更新通知
- **描述**: 支持RSS订阅，预留微信推送接口

### AI辅助功能
- **目的**: 提升创作和阅读体验
- **描述**: 包含文章排版优化、智能推荐、联网搜索、功能代码生成等AI能力

### 多用户管理
- **目的**: 支持多人协作管理
- **描述**: 管理员、编辑、作者、订阅用户、访客等角色权限管理

### 后台管理
- **目的**: 优雅的内容和用户管理界面
- **描述**: 基于Vue.js + Element Plus的现代化后台管理界面
